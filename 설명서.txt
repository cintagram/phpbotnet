iot pma 봇넷 사용 설명서

1. python 파일을 실행
2. bots << 봇 불러오는 명령어
3. help << 메소드 리스트

봇 만드는법

1. https://sprinthost.ru/ 사이트에 겨들어간다
2. http://ruu.kr/ 해당 사이트도 겨들어간다
3. 1회용 이메일을 생성 후 1번 사이트에 가입한다
4. 1번 사이트에서 파일 관리자에 들어간뒤 php파일을 넣는다
5. http://a0751305.xsph.ru/index.php 이러한 형식으로 bots.txt에 봇을 넣는다

참고 사항

1. crypto.php는 암호화된 페이로드 php파일이다
2. index.php는 일반 페이로드 php파일이다
